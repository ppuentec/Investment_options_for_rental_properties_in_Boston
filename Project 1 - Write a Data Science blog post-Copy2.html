Section 1 : Business Understanding¶

In this project, I will be using the Boston AirBnB dataset taken from Kaggle. I will be using this information to predict the revenue of a small rental property in the Boston area. Through this project, I will answer the following questions:

  • What is the expected occupancy rate of a rental property in Boston based on a rental price?
  • What is the Average Daily Rate (expected revenue) of a rental property in the Boston area?
  • Try to determine if a small apartment or a small house is a better option for investment and future rental in the Bostor area. This analysis can be the foundation for a more sophisticated to predict rental income for a small investments in the Boston area.

Section 2 : Data Understanding¶

The initial dataset was taken from the 'listings.csv' file. This file contains 3,585 listed properties with 95 columns with many, many features for each property such as ratings, pictures, prices, and many more.

In this analysis, I will reduce this dataset to only 14 columns to provide me the necessary information to answer previous questions. I developed this work using a classroom Jupyter notebook.

# The following function replaces a categorical column with several dummy columns that contain the categorical values.

def create_dummy_df(df, cat_cols, dummy_na):
    for col in cat_cols:
        try:
            df = pd.concat([df.drop(col, axis=1), pd.get_dummies(df[col], prefix=col, prefix_sep='_', drop_first=False, dummy_na=dummy_na)], axis=1)
        except:
            continue
        return df
